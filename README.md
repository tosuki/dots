# Dots
Here you can find my configuration file of:
- i3wm
- mate-terminal
- my cursors
- fonts

## Quick setup
Although it may work in other distros, the initial purpose of this script is solely to make my life easier
```sh
git clone http://github.com/tosuki/dots ./
cd dots
sudo ./install.sh
```

## Keybinds
### gaps
- Alt + Control + Up - Increase inner gaps
- Alt + COntrol + Down - Decrease inner gaps

### Navigation
- Alt + (1-7) - Switch workspaces
- Alt + Shift + (1-7) Move client between workspaces

- Alt + Left - Select the client on the Left
- Alt + Right - Select the client on the Right
- Alt + Up - Select the client above
- Alt + Down - Select the client on the Down

- Alt + Shift + Left - Swap place with the client on the left
- Alt + Shift + Right - Swap place with the client on the right
- Alt + Shift + Up - Swap place with the client above
- Alt + Shift + Down - Swap place with the client below

- Alt + Space - Change focus between floating and tiling clients

- Alt + V - Split in vertical orientation
- Alt + H - Split in horizontal orientation
### Modes
- Alt + Shift + Space - toggle floating mode
- Alt + R - toggle resize mode
- Alt + E - toggle split layout
- Alt + W - toggle tabbed layout
- Alt + S - toggle stacking layout

### General
- Alt + z - Runs boomer
- Alt + D - Runs dmenu
- Alt + Shift + E - exit i3
- Alt + Shift - Q - close client
- Alt + Return - Open mate-terminal
